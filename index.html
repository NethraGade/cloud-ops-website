Program Curriculum

An 8-week intensive journey from basics to mastery.

Week 1-2: Foundations of Cloud & DevOps

• Intro to Cloud Computing (Azure/AWS)
• Linux Essentials for DevOps
• Git & Version Control Mastery
• Project: Hosting a static website on Azure Storage

Week 3-4: CI/CD Pipelines with Azure DevOps

• Azure Repos & Branching Strategies
• Writing YAML Pipelines (Build & Release)
• Integrating Unit Tests & Code Quality Checks
• Project: Automated Build Pipeline for a Node.js App

Week 5: Containerization (Docker)

• Docker Fundamentals (Images, Containers, Registry)
• Dockerfile Best Practices
• Azure Container Registry (ACR)
• Project: Containerizing a Microservice Application

Week 6: Infrastructure as Code (Terraform/Bicep)

• Principles of IaC
• Provisioning Azure Resources with Terraform
• State Management & Modules
• Project: Deploying full infra stack with one command

Week 7: Kubernetes & Orchestration

• Kubernetes Architecture (Pods, Services, Ingress)
• Azure Kubernetes Service (AKS)
• Helm Charts
• Project: Deploying the Microservice to AKS

Week 8: Monitoring & Capstone

• Azure Monitor & Application Insights
• Logging & Alerting
• Final Capstone: End-to-End DevOps Implementation for an E-commerce Platform
